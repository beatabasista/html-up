This is a simple javascript library can be used to boost up the ability of html to do sum mathematical calculations. Like addition subsraction and other thing slike calculating the value of trigonometric ratios..


Note : This is just a prototype and can be continued building up....
Some tags are defined using the Script.js file, their syntax is given below..
ADD tag:
<add> Values saparated by '+' operator</add> ---> <add>2+5+65+10</add> 
Substraction tag:
<subs> Values saparated by '-' operator</subs> ---> <subs>2-5-65-10</subs>
Multiplication tag:
<mult> Values saparated by '*' operator</mult> ---> <mult>2*5*65*10</mult>
Division tag:
<divide> Values saparated by '/' operator</divide> ---> <add>2/5/65/10</divide>
Exponent tag:
<power>number,Power</power> ---> <power>2,3</power> ---> 8
largest Tag:
<largest>values saperated by ',' </largest> ---> <largest>10,67,34,23,69</largest> ---> 69
 Smallest Tag:
<smallest>values saperated by ',' </smallest> ---> <smallest>10,67,34,23,69</smallest> ---> 10
